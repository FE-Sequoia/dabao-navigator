# TabHome 大宝标签管理器 - 本地标签管理插件

## 项目简介

TabHome 大宝标签管理器是一个开源的 Chrome 浏览器标签管理插件，用于替换默认的新标签页，提供分类管理标签的功能。

### 核心功能

- **新标签页替换**：安装插件后，所有新打开的标签页默认加载插件的首页。
- **标签分类管理**：按分类展示用户添加的网页链接。
- **默认数据**：内置常用网站的默认数据。
- **添加/编辑/删除标签**：支持对单个标签进行操作。
- **分类管理**：支持新增、重命名、删除分类。
- **导入/导出**：支持导出和导入 JSON 数据文件。
- **纯本地存储**：所有数据仅存储在本地，不向任何服务器发送。

## 技术栈

- Chrome Extension (Manifest V3)
- HTML + CSS + JavaScript
- 纯本地开发，不依赖后端服务器

## 安装方法

1. 下载本项目的源代码或打包好的 `.zip` 文件。
2. 解压文件到本地目录。
3. 打开 Chrome 浏览器，进入 `chrome://extensions/` 页面。
4. 开启右上角的「开发者模式」。
5. 点击「加载已解压的扩展程序」，选择解压后的目录。
6. 安装完成后，打开新标签页即可看到TabHome 大宝标签管理器的界面。

## 使用说明

### 添加标签
1. 点击底部操作栏的「添加标签」按钮。
2. 填写标签名称、URL、图标URL（可选），并选择分类。
3. 点击「保存」按钮。

### 编辑标签
1. 鼠标悬停在标签卡片上，点击「编辑」按钮。
2. 修改标签信息。
3. 点击「保存」按钮。

### 删除标签
1. 鼠标悬停在标签卡片上，点击「删除」按钮。
2. 在确认对话框中点击「确认」按钮。

### 管理分类
1. 点击分类标题右侧的「重命名」按钮可以修改分类名称。
2. 点击分类标题右侧的「删除」按钮可以删除分类（会同时删除该分类下的所有标签）。
3. 点击底部操作栏的「添加分类」按钮可以新增分类。

### 导入/导出数据
1. 点击底部操作栏的「导出JSON」按钮可以导出当前所有标签为 `tabs.json` 文件。
2. 点击底部操作栏的「导入JSON」按钮可以选择并导入外部的 `tabs.json` 文件（会完全覆盖现有数据）。

### 搜索标签
在顶部的搜索框中输入关键词，可以实时搜索标签名称和URL。

## 数据结构

标签数据以 JSON 格式存储在 `chrome.storage.local` 中，结构如下：

```json
{
  "categories": [
    {
      "id": "1",
      "name": "工作",
      "tabs": [
        {
          "id": "1-1",
          "name": "GitHub",
          "url": "https://github.com",
          "icon": "https://github.com/favicon.ico"
        }
      ]
    }
  ]
}
```

## 开发指南

### 项目结构

```
dabao-navigator/
├── manifest.json        # Chrome Extension 配置文件
├── index.html           # 新标签页页面
├── script.js            # 核心功能逻辑
├── tabs.json            # 默认标签数据
├── icons/               # 插件图标目录
└── README.md            # 项目说明文档
```

### 开发步骤

1. 克隆项目到本地。
2. 修改代码。
3. 在 Chrome 扩展管理页面点击「重新加载」按钮查看修改效果。
4. 测试功能。
5. 打包发布。

## 打包方法

1. 确保所有文件都已准备就绪。
2. 将项目目录压缩为 `.zip` 文件。
3. 发布到 GitHub Release。
