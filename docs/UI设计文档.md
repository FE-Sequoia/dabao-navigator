# TabHub - 本地标签管理插件 UI设计文档

## 1. 设计风格

### 1.1 整体风格
- **风格**：简洁、现代、科技、酷炫
- **参考**：ant-design pro风格，类似浏览器书签页
- **配色**：
  - 主色调：深蓝色 (#1890ff)
  - 辅助色：浅蓝色 (#40a9ff)
  - 背景色：浅灰色 (#f0f2f5)
  - 卡片背景：白色 (#ffffff)
  - 文字颜色：深灰色 (#333333)、中灰色 (#666666)、浅灰色 (#999999)
- **字体**：系统默认字体，优先使用无衬线字体
- **阴影**：轻微的卡片阴影，增强层次感

## 2. 页面布局

### 2.1 整体布局
- **顶部**：插件标题 + 搜索框
- **主体**：多列卡片式布局，按分类分组展示标签卡片
- **底部**：操作栏（添加标签、导入/导出 JSON、设置）

### 2.2 顶部导航栏
- **高度**：60px
- **布局**：左侧为插件标题，右侧为搜索框
- **插件标题**：
  - 文字："TabHub"
  - 字体大小：20px
  - 字体权重：600
  - 颜色：#1890ff
- **搜索框**：
  - 宽度：300px
  - 高度：36px
  - 边框：1px solid #d9d9d9
  - 圆角：4px
  - 背景：白色
  -  placeholder："搜索标签..."

### 2.3 主体内容区
- **布局**：响应式多列布局，根据屏幕宽度自动调整列数
  - 大屏幕：4列
  - 中屏幕：3列
  - 小屏幕：2列
- **分类标题**：
  - 字体大小：18px
  - 字体权重：600
  - 颜色：#333333
  - 间距：16px
- **标签卡片**：
  - 宽度：200px
  - 高度：120px
  - 边框：1px solid #e8e8e8
  - 圆角：8px
  - 背景：白色
  - 阴影：0 2px 8px rgba(0, 0, 0, 0.09)
  - 悬停效果：阴影加深，轻微上浮
  - 内部布局：
    - 图标：顶部居中，大小48px
    - 名称：图标下方，字体大小14px，颜色#333333
    - 操作按钮：悬停时显示在卡片右上角

### 2.4 底部操作栏
- **高度**：60px
- **布局**：左侧为添加标签按钮，右侧为导入/导出/设置按钮
- **按钮样式**：
  - 主按钮（添加标签）：
    - 背景：#1890ff
    - 文字：白色
    - 圆角：4px
    - 悬停：#40a9ff
  - 次要按钮（导入/导出/设置）：
    - 背景：白色
    - 文字：#1890ff
    - 边框：1px solid #1890ff
    - 圆角：4px
    - 悬停：#e6f7ff

## 3. 交互设计

### 3.1 标签卡片交互
- **点击**：在新标签页打开对应URL
- **鼠标悬停**：
  - 卡片轻微上浮，阴影加深
  - 显示编辑/删除按钮
- **编辑按钮**：点击后弹出编辑标签对话框
- **删除按钮**：点击后弹出确认删除对话框

### 3.2 分类管理交互
- **分类标题悬停**：显示分类管理按钮（新增/重命名/删除）
- **新增分类**：点击后弹出新增分类对话框
- **重命名分类**：点击后弹出重命名分类对话框
- **删除分类**：点击后弹出确认删除对话框

### 3.3 导入/导出交互
- **导出JSON**：点击后生成并下载tabs.json文件
- **导入JSON**：点击后弹出文件选择对话框，选择后读取并导入数据

### 3.4 搜索交互
- **输入时**：实时过滤标签，显示匹配的标签
- **清空输入**：显示所有标签

## 4. 对话框设计

### 4.1 添加/编辑标签对话框
- **标题**："添加标签"或"编辑标签"
- **表单字段**：
  - 名称：文本输入框
  - URL：文本输入框
  - 图标：文本输入框（可输入图标URL）
  - 分类：下拉选择框
- **按钮**：
  - 取消：灰色按钮
  - 保存：蓝色按钮

### 4.2 分类管理对话框
- **新增分类**：
  - 标题："新增分类"
  - 字段：分类名称
  - 按钮：取消、确定
- **重命名分类**：
  - 标题："重命名分类"
  - 字段：新分类名称
  - 按钮：取消、确定
- **删除分类**：
  - 标题："删除分类"
  - 内容："确定要删除该分类吗？删除后该分类下的所有标签也会被删除。"
  - 按钮：取消、删除

### 4.3 确认对话框
- **标题**："确认操作"
- **内容**：根据操作类型显示不同的确认信息
- **按钮**：取消、确认

## 5. 响应式设计

### 5.1 屏幕尺寸适配
- **大屏幕**（> 1200px）：4列标签卡片
- **中屏幕**（768px - 1200px）：3列标签卡片
- **小屏幕**（< 768px）：2列标签卡片

### 5.2 移动端适配
- **顶部导航栏**：搜索框宽度自适应
- **主体内容**：2列标签卡片
- **底部操作栏**：按钮垂直排列

## 6. 视觉效果

### 6.1 动画效果
- **标签卡片悬停**：平滑过渡的阴影和位置变化
- **对话框弹出/关闭**：淡入淡出效果
- **添加/删除标签**：平滑的添加/删除动画

### 6.2 图标设计
- **使用**：网站自带的favicon作为标签图标
- **默认图标**：当网站没有favicon时，使用默认图标

### 6.3 色彩方案
- **主色调**：#1890ff（蓝色）- 代表科技、信任
- **辅助色**：#40a9ff（浅蓝色）- 用于悬停效果
- **背景色**：#f0f2f5（浅灰色）- 提供舒适的视觉体验
- **卡片背景**：#ffffff（白色）- 突出内容
- **文字颜色**：#333333（深灰色）、#666666（中灰色）、#999999（浅灰色）- 层次分明
